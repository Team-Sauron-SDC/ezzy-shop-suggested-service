# Suggested Module

##CRUD Operations

| HTTP Verb |           Endpoint         |            Action            |
|-----------| -------------------------- | ---------------------------- |
| **POST**  |         /products/         |  CREATE a new item into DB   |
| **GET**   |       /products/:id        |  READ data and return data   |
| **PATCH** |       /products/:id        |  UPDATE item with new image  |
| **DELETE**|       /products/:id        |  DELETE item based on ID     |
